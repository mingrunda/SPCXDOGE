# SPCXDOGE Website V1

English-only meme landing page.

## Sections

- Hero visual
- Meme Gallery
- Live Data
- Why Hold LP
- Tokenomics
- Genesis Shareholder
- Roadmap
- Community
- Enter DApp

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Replace links

Edit `app/page.tsx` and replace placeholder Twitter / Telegram / DApp links.

## Replace logo

Replace `public/logo.png` with your final transparent logo.
