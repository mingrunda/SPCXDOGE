# SPCXDOGE Meme Website V3.5 Responsive

Responsive version for desktop, tablet and mobile.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

- Framework Preset: Next.js
- Build Command: npm run build
- Output Directory: out

## Changes in V3.5

- Desktop/mobile adaptive layout
- Mobile navbar simplified
- Hero title fixed on desktop and stacked on mobile
- Story cards: 4-column desktop, 2-column tablet, 1-column mobile
- Tokenomics: 2-column desktop, 1-column mobile
- Fan Art: marquee desktop, swipe cards mobile
- Live Data: responsive cards
- Genesis and Roadmap mobile layout optimized
- Better section spacing and readability
