# SPCXDOGE Meme Website V3.3

V3.3 refinement:

- Removed Rewards from navbar; rewards are merged into Tokenomics
- Added official Twitter and Telegram links
- Added 8 original fan-art cards
- Improved readability and spacing
- Darker black/gold balance
- Genesis cards upgraded from emoji to badge style
- Section scroll margin fixed for sticky navbar

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

Upload this folder to GitHub, import the repo in Vercel, and deploy with default Next.js settings.
